###栈（Stack）抽象的数据结构
**什么是栈？**
**栈**是一种遵从``后进先出（Last Input First Output）的有序集合``。
对于栈的操作，**只能在一端进行**，这一端被称之为栈顶，而另一端被称之为栈底。例如：生活中的书堆的垂直摆放，在顶端进行存取操作，不允许中间存取（可能会发生意向不到的结果）。。。
**form wikipedia**
栈：被认为是线性结构，或者更加抽象的顺序集合。
